Créer une base Firebase (gratuit, 5 min)
  1. Allez sur console.firebase.google.com et créez un projet (désactivez Google Analytics si souhaité).
  2. Dans le menu gauche → Firestore Database → Créer une base de données → Mode test (à sécuriser ensuite).
  3. Dans Règles, collez ces règles puis publiez :
    rules_version = '2'; service cloud.firestore { match /databases/{db}/documents { match /households/{code} { allow read, write: if code.size() == 6; } } }
  4. Retournez dans les Paramètres du projet (roue dentée) → Vos applications → ajoutez une appli Web (</>).
  5. Copiez l'objet firebaseConfig et collez-le dans le champ ci-dessus.
  6. Si vous utilisez GitHub Pages, ajoutez votre domaine dans Authentication → Domaines autorisés.
